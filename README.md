# Lucky draw by React
* React 17.0.1
* React Redux 7.2.2
* Next 10.0.8
* Typescript 4.2.3

## Requirements
1. Use with **cshimegi/lucky-draw-api**

## Installation from scratch
##### Notification
If you want to install from scratch, please refer to branch **master**.

##### Steps
* Step 1
`git clone https://github.com/cshimegi/lucky-draw-gui.git`

* Step 2
```bash
$ cd /to/git/clone/directory
$ cd /to/project/top
$ npm install
```

* Step 3
```bash
$ npm run build
$ npm run start
```

## Access to Django WEB
After server is ready, you can access WEB page from browser.

`http://localhost:3000`
